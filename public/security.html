Lineage Theatre Security

Lineage Theatre is currently a public static preview. Customer files and project data are stored in the visitor's browser first using local browser storage. Do not upload sensitive real family records until production authentication, encrypted server storage, and signed upload workflows are connected.

Security reports and launch questions can be sent to info@lineagetheater.com.

The public preview uses a strict content security policy, disables high-risk browser permissions, avoids third-party scripts, and keeps payment, email, video rendering, and publishing integrations behind adapter boundaries until production services are ready.
